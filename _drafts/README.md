# Drafts

Place draft articles here. Files in `_drafts` are **not published** to the live site.

To preview drafts locally, run:

```bash
jekyll serve --drafts
```

Draft filenames don't need a date prefix (e.g. `my-new-article.md` instead of `2026-04-01-my-new-article.md`). Jekyll will use the file's last-modified date when previewing.
